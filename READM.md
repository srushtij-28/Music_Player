# Music Player (Pygame)

A simple music player made using Python and Pygame. It plays a song and
shows a rotating disc animation.

## Features

-   Play music
-   Pause / Resume
-   Stop music
-   Rotating disc animation

## Controls

-   SPACE -- Pause / Resume
-   S -- Stop
-   ESC -- Exit

## How to Run

1.  Install pygame\
    pip install pygame
2.  Keep main.py, dvd.png, and song.mp3 in the same folder
3.  Run: python main.py

## Requirements

-   Python 3
-   Pygame
